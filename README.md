Prerequisiti: 

-python3 -m pip install --no-cache-dir numpy pandas matplotlib

-pip3 install flask


Istruzioni per l'esecuzione:

1) aprire la cartella dove si trova "start_project.sh"

|--> copiare i file all'interno della cartella "components" in una cartella dallo stesso nome

|--> copiare i file dell'interfaccia flask in una cartella denominata "flaskInterface"

2) eseguire "start_project.sh"

3) seguire le INFO stampate sul terminale per testare i vari scenari di attacco
